#### 📌 Projetos em Destaque

🤖 **[Sophos Kodiak — Assistente de Análise de Dados com IA](https://github.com/jpbernucio/sophos_kodiak)**  
Aplicativo mobile que converte perguntas em linguagem natural em consultas SQL utilizando LLM (Google Gemini), permitindo análise de métricas de vendas através de chatbot e dashboard analítico.

`Flutter` `Python` `Supabase` `Google Gemini API` `Text-to-SQL`

- Pipeline Text-to-SQL com LLM para análise de dados em linguagem natural
- Interface com chatbot (histórico de conversas) e dashboard com filtros
- Integração com banco de dados para análise de métricas de vendas

<br>

📄 **[Sistema de Automação de NFe](link-do-repo)**

`Python` `SQLite` `Redis` `MinIO` `Docker`

- Validação e processamento automatizado de arquivos XML (NFe)
- Arquitetura com cache (Redis) e armazenamento de arquivos (MinIO)
- Containerização com Docker para padronização de ambientes

<br>

📊 **[Sistema de Observabilidade e Monitoramento](link-do-repo)**  
Implementação de stack completa de observabilidade para centralização de logs e monitoramento de infraestrutura AWS.

`Promtail` `Loki` `Grafana` `Prometheus` `AWS Lightsail` `Docker`

- Pipeline de logs centralizado (Promtail → Loki → Grafana)
